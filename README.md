# Snubes

## Installation Steps

1. `npm install`
2. `npm start`

### To Run Test Cases

`npm test`

## Following are the tech stack used for this project

1. webpack
2. babel for transpiling
3. React JS(v16.0.00)
4. Redux
5. Jest(for unit testing)
6. React DnD(Drag and Drop library)
